## Overview
This portfolio was built to act as the central hub for my digital presence, moving away from standard templates into a deeply customized, high-fidelity React application. It uses a custom **"Dark Room"** aesthetic built with modern Tailwind CSS v4 and Framer Motion, utilizing scroll-driven animations, complex layered glassmorphism, and responsive tech-dashboard components.

**Live Site:** [wablevaibhav.github.io](https://wablevaibhav.github.io)

## Tech Stack 
- **Framework:** React 19 + Vite
- **Styling:** Tailwind CSS v4 (with custom `@theme` variables and utility abstractions)
- **Animations:** Framer Motion (Scroll animations, variants, layout transitions)
- **Icons:** React Icons (`react-icons/fi`)
- **Typography:** Inter & JetBrains Mono

## Project Status & Architecture
- **Data Source:** All content is decoupled and driven by a central payload located in `/src/data/portfolioData.js`. 
- **Design System:** Utilizes raw CSS variables generated from the base index for dynamic, glowing components (see `index.css`).
- **Interactive Timelines:** Includes custom horizontal scroll-clipped pills and timeline components.

## Running Locally

To clone and run this application locally, you'll need Node.js installed.

```bash
# 1. Clone the repository
git clone https://github.com/wablevaibhav/wablevaibhav.github.io.git

# 2. Enter the directory
cd wablevaibhav.github.io

# 3. Install dependencies
npm install

# 4. Start the Vite development server
npm run dev
```
